# 基礎概述

Material Design 經由使用者介面端、平面以及元件定義了何為品質。使用可從廣泛和詳細角度解決設計問題的基礎來設計和製定應用程序的構建策略。

## 排版、顏色與其他

本節描述了材質環境與排版，其可能的交互作用以及顏色、形狀、動作在品質上的統一表現。
